Date Type Introduction

String
A String is a collection of letters and numbers. The data of the string are placed inside of double quotes ""

Example:
var a = "My name is Daniel"
var a = "Today's date is: 2024/07/11"

Integer
An integer is a number that has no decimal. Any value after the decimal is dropped.

Example:
var a = 3
var number = 12.34;
var integer_value = parseInt(number);
// the variable integer_value will contain 12

Float
A float or floating point number is number with decimal

Example:
var a = 3.14159

Array or List
An array or a list is a way to store data squentially, but users are able to access any data by its index -- reading, writing and inserting.
Think of a stack of card. Normally, you put or pull a card from the top of deck or bottom of the deck.
To create an array, the data are placed between square brackets []

Example:
var a = [ "a", "b", 2, 4, 6 ];
var b = new Array("a", "b", 2, 4, 6);

For a deck of card, above as an example. Let say your deck of card originally the King is on top. Then you place the Queen on top of the deck. That's storing data into the array.
It also works on the reverse. You remove the King from the deck. That's reading the data from the array.
Also, on the oppersite end. Data can be store and read from the bottom of the array. -- Taking a card from the bottom of the deck.

Dictionary, Map or Associative Array
A dictionary, map and associative array is to store data into a key and value pair. Like a book of dictionary, you can look up a word to get it's definition.
A dictionary/map/associated array data are placed in a pair of brances {}.

Example:
var a = { };
or
var b = { "1st grade": 33, "2nd grade": 22, "3rd grade": 23, "4th grade": 30, "5th grade": 33, "6th grade": 26 }
